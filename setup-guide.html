⚡ Installation

Ein Befehl installiert alles automatisch

🚀 All-in-One Installer

Der Installer erkennt dein System und installiert automatisch alles was fehlt: Homebrew (Mac), Node.js, yt-dlp, ffmpeg und MindVault.

  1. 1
    Terminal öffnen Mac: Spotlight (⌘ + Leertaste) → "Terminal" eingeben
    Windows: Startmenü → "cmd" eingeben
  2. 2
    Installer starten Navigiere zum MindVault-Ordner und starte den Installer:
    cd ~/Downloads/MindVault
    chmod +x install.sh
    ./install.sh
    cd %USERPROFILE%\Downloads\MindVault
    install.bat
  3. 3
    Lehne dich zurück Der Installer macht den Rest. Du siehst den Fortschritt live im Terminal:
    Terminal — install.sh

🤖 Telegram Bot erstellen

Erstelle deinen persönlichen Bot um Links an MindVault zu senden

📱 Bot bei BotFather erstellen

  1. 1
    BotFather öffnen Öffne Telegram und suche nach @BotFather — das ist Telegrams offizieller Bot-Ersteller.
    🤖
    BotFather
    verified bot
    Tippe auf Start oder sende /newbot
  2. 2
    Neuen Bot erstellen Sende den Befehl:
    /newbot
  3. 3
    Namen vergeben BotFather fragt nach einem Anzeigenamen und einem Benutzernamen.
    Alright, a new bot. How are we going to call it? Please choose a name for your bot.
    MindVault
    Good. Now let's choose a username for your bot. It must end in 'bot'. Like this, for example: TetrisBot or tetris_bot.
    mein_mindvault_bot
    💡 Der Benutzername muss auf bot enden und einzigartig sein. Zum Beispiel: marco_mindvault_bot
  4. 4
    Token kopieren BotFather gibt dir einen API Token — das ist der Schlüssel für deinen Bot.
    Done! Congratulations on your new bot.

    Use this token to access the HTTP API:
    7123456789:AAH1234abcd5678efgh-XYZTOKEN

    Keep your token secure and store it safely.
    ⚠️ Wichtig: Teile diesen Token mit niemandem! Er gibt vollen Zugriff auf deinen Bot.

⚙️ Token in MindVault eintragen

Starte MindVault und trage den Token in den Settings ein:

  1. 1
    MindVault starten
    ./start.sh
    start.bat
  2. 2
    Öffne localhost:3000 im Browser
  3. 3
    Settings öffnen → Telegram Bot Token einfügen Klicke auf das Zahnrad-Icon (⚙️) rechts oben und füge den Token ein.
    MindVault — Settings
    Telegram Bot Token
    7123456789:AAH1234abcd5678...
    Save
    ✓ Bot connected: @mein_mindvault_bot

🌐 Webhook mit ngrok verbinden Für lokale Nutzung

Telegram muss deinen lokalen Server erreichen können. Dafür brauchst du ngrok (oder einen Server mit öffentlicher IP).

  1. 1
    ngrok installieren Erstelle ein kostenloses Konto auf ngrok.com und lade es herunter.
    # Mac (Homebrew)
    brew install ngrok
  2. 2
    ngrok starten
    ngrok http 3001
    Du bekommst eine URL wie https://abc123.ngrok.io
  3. 3
    Webhook wird automatisch gesetzt MindVault erkennt die ngrok-URL automatisch beim Starten und setzt den Telegram Webhook.
    ✓ Jetzt kannst du Links an deinen Bot senden und sie erscheinen in MindVault!

🤖 AI API verbinden

Optional: Aktiviere automatisches Tagging mit Anthropic Claude

🔑 Anthropic API Key erstellen

  1. 1
    Anthropic Console öffnen Gehe zu console.anthropic.com und erstelle ein Konto (falls noch nicht vorhanden).
  2. 2
    API Key generieren Gehe zu Settings → API Keys → Create Key
    Anthropic Console — API Keys
    API Keys
    + Create Key
    MindVault
    sk-ant-api03-...xxxxx
    Active
  3. 3
    Key kopieren Kopiere den angezeigten Key. Er beginnt mit sk-ant-...
    ⚠️ Wichtig: Der Key wird nur einmal angezeigt! Kopiere ihn sofort.
  4. 4
    In MindVault eintragen Öffne MindVault Settings → AI API Key → Einfügen → Speichern
    MindVault — Settings
    AI API Key (Anthropic)
    sk-ant-api03-xxxxxxx...
    Save
    ✓ AI connected — Auto-tagging enabled
